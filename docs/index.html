О программе

AudioVolumeNormal — это программа для увеличения громкости аудио сигнала в медиа файлах. Программа разработана как кроссплатформенная, поддерживаемые ОС Windows и Linux, версии 64 бит.

Опробована на Windows 7, 10, 11, Linux Ubuntu 18.04.

Возможности программы

  • Поддержка популярных форматов: MP4, AVI, MKV, TS.
  • Пакетная обработка нескольких файлов.
  • Увеличить уровень аудио сигнала до "нормы" (0dB).
  • Сконвертировать аудио поток (дорожку) из формата 5.1 (6.1, 7.1 ...) в stereo.
  • Изменить аудио кодек на AAC или AC3.
